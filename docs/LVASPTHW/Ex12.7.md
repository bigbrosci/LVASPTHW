# Ex12.7 POSCAR 坐标格式转换脚本使用说明

本脚本 `reformat.py` 用于在 VASP 的 POSCAR 文件中，实现**Cartesian（笛卡尔坐标）** 和 **Direct（分数坐标）** 之间的相互转换。

如果对你有帮助，请我喝咖啡，打赏码在下载的脚本压缩包里面。下载：[reformat.zip](../Downloads/reformat.zip)

## 📌 功能简介

-   只支持命令行运行，并且安装了ase。(没有安装请看本书前面的章节) 
-   支持输入标准 POSCAR 文件
-   输出两种格式：
    -   Cartesian（笛卡尔坐标）
    -   Direct（分数坐标）
-   自动生成新的 POSCAR 文件，不覆盖原文件

------------------------------------------------------------------------

## 📂 文件准备

### 方法一：放在当前目录（一次性的简单方式）

确保当前目录下包含：

POSCAR
reformat.py

✔ 缺点：

* 每次都要把脚本复制到POSCAR所在的目录。

------------------------------------------------------------------------

### 方法二（推荐）：加入系统路径（避免每次都要复制）

建议将脚本放入 `~/bin`
目录，这样可以在任意路径下直接调用脚本，而无需将其复制到每个工作目录。

#### 步骤：

```bash
mkdir -p ~/bin
cp reformat.py ~/bin/
chmod +x ~/bin/reformat.py
```

#### ✔ 优点：

-   不需要在 POSCAR 所在目录放置脚本
-   可以在任意目录直接运行


------------------------------------------------------------------------

## ⚙️ 使用方法

### 1️⃣ 转换为 Cartesian 坐标的命令：

reformat.py POSCAR c

输出文件：

POSCAR_cartesian

------------------------------------------------------------------------

### 2️⃣ 转换为 Direct 坐标的命令

reformat.py POSCAR d

输出文件：

POSCAR_direct

------------------------------------------------------------------------

## 🧠 参数说明

  参数     含义
-------- ------------------
  POSCAR   输入文件
  c        转换为 Cartesian
  d        转换为 Direct

------------------------------------------------------------------------
